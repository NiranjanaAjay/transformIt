📋 TROUBLESHOOTING
------------------

Q: "npm command not found"
A: Install Node.js from nodejs.org → restart terminal

Q: Port 5173 already in use
A: Kill the process or wait a moment → npm run dev again

Q: Styling not appearing
A: Clear browser cache (Ctrl+Shift+Delete) → refresh page

Q: Changes not showing
A: Check console for errors (F12) → restart dev server

Q: PowerShell errors
A: Use Command Prompt (cmd) instead of PowerShell
